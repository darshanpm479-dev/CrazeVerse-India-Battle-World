# Local Prototype Scope

## Must Work Locally First

- Guest login
- Local profile
- Local save
- Local coins
- Local XP
- Local rank
- Bot players
- Squad bots
- Rival squad bots
- Local matchmaking simulation
- Local leaderboard mock
- Local shop mock
- Local custom room
- Local mission database
- Local rewards
- Local settings

## Must Not Block Prototype

- No Firebase dependency
- No live backend dependency
- No live payment dependency
- No online multiplayer dependency
- No OpenAI API dependency
- No Google login dependency

## MVP Proof

The MVP must prove app opening, guest profile, home lobby, mode select, Training Ground, Ghost Hunt loop, rewards, local save, and basic multi-mode navigation.
