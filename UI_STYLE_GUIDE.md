Use Canva Deck 1 to Deck 5 as the main design reference. Figma Step 2 is partially completed and pending due tool limit.

## UI Rules

- Android mobile-first.
- Unity Canvas reference: 1080 x 2400 portrait.
- Figma frame references: 360 x 800 and 412 x 915.
- Use 8dp spacing grid.
- Minimum touch target: 48dp.
- Bottom safe area: 34px.
- Top safe area: 48px.
- Big readable buttons.
- Premium game cards.
- Separate lobby UI anchors and HUD UI anchors.
- Avoid text baked into images.
- Export icons as SVG.
- Export raster art as PNG or WebP.
- Use sliced sprites for buttons and panels.
- Build Unity UI as data-driven prefabs.
- Must feel premium, cinematic, Indian, adventurous, and high contrast.
