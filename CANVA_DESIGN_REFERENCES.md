# Canva Design References

Canva is the primary visual direction tool for CrazeVerse India Battle World.

## Deck 1: Brand + Main App UI

Purpose: Logo direction, app icon, splash screen, home lobby, profile page, and mode selection.

## Deck 2: Core Game Modes

Purpose: Royale lobby, Ghost Hunt Squad main page, mission list, Ghost Hunt HUD, team lobby, and survival lobby.

## Deck 3: Extra Modes + Social

Purpose: Race lobby, Custom Room page, Training Ground page, Friends page, Squad lobby, and Clan page.

## Deck 4: Systems + Rewards

Purpose: Shop, rewards, battle pass, leaderboard, settings, report page, and map download manager.

## Deck 5: Concept Boards + Pitch Summary

Purpose: India-inspired map boards, character roles, loadout and vehicle board, 126+ mission overview, and production roadmap summary.

## Developer Usage

Unity developers must use these decks as visual reference only. Rebuild UI in Unity as responsive, data-driven prefabs. Do not bake important text into images.
