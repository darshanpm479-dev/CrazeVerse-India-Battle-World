# CrazeVerse India Battle World

Android-first Unity 3D URP mobile game planning repo for a premium Indian-inspired battle-horror adventure game.

## Current Status

- Step 0: Connected apps and local-first architecture confirmed.
- Step 1: Canva Deck 1 to Deck 5 completed as visual direction.
- Step 2: Figma file created, full handoff pending due tool limit.
- Step 3: GitHub planning files added.

## Game Modes

1. Battle Royale - 100-player architecture, local bots first.
2. Ghost Hunt Squad - flagship multi-squad mission mode.
3. Team Deathmatch - 5v5 fast combat.
4. Zombie Survival - 4-player wave survival design.
5. Race Battle - vehicle fight racing.
6. Custom Room - private friend matches.
7. Training Ground - practice for guns, movement, vehicles, ghosts, and zombies.

## Local-First Rule

The first playable build must not depend on Firebase, Razorpay, Photon, PlayFab, Nakama, OpenAI API, Google login, real backend, real payment, or real multiplayer.

The prototype must use guest login, local profile, local save, coins, XP, rank, bots, squad bots, enemy squad bots, local matchmaking simulation, mock leaderboard, mock shop, mock rooms, and local mission database.
