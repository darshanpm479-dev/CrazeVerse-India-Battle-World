# Unity Setup Guide

Use Unity 3D URP for an Android-first mobile game project.

## Project Direction

- Engine: Unity 3D URP.
- Language: C# scripting.
- Target: Android mobile first.
- Gameplay target: mainly landscape.
- Menu target: landscape first; portrait support can be planned only if needed.
- Prototype rule: local working prototype first.
- No Firebase, backend, payment, multiplayer server, or OpenAI API dependency for this step.
- Use Canva Deck 1 to Deck 5 and UI_STYLE_GUIDE.md for visual direction.
- Figma handoff is pending and not blocking the Unity structure.
- Keep the project clean, modular, and ready for later services.
