# Local Unity Opening Checklist

Use this checklist when opening the project on the local computer.

## Install Tools

1. Install Unity Hub.
2. Install Unity Editor.
3. Add Android Build Support.
4. Add Android SDK module.
5. Add Android NDK module.
6. Add OpenJDK module.

## Open Project

1. Clone or download the GitHub repo.
2. Open Unity Hub.
3. Choose the project folder.
4. Open the project.
5. Let Unity import assets completely.
6. Switch platform to Android.
7. Open the Console.
8. Check all errors.
9. If errors exist, fix scripts first.
10. If no errors exist, continue scene setup planning.

## Do Not Rush

Do not create APK or add online services until the Console has zero compile errors.
