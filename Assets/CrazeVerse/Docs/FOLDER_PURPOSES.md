# Folder Purposes

- Art: logo drafts, icons, key art, concept images.
- Audio: music and voice placeholder assets.
- Characters: player, bot, and role assets.
- Environments: environment pieces and scene props.
- Maps: map layout assets and greybox references.
- Modes: mode-specific assets and configuration.
- UI: menu, lobby, HUD, and mobile interface assets.
- Scripts: C# source folders grouped by system.
- Data: local JSON, ScriptableObject data, and static tables.
- Prefabs: reusable Unity prefabs.
- Scenes: Unity scene files.
- VFX: visual effect assets.
- SFX: short sound effect assets.
- Settings: URP, input, graphics, quality, and project settings notes.
- Resources: Unity Resources loaded assets when required.
- StreamingAssets: local static data that must ship with the app.
- Tests: test plans and later Unity test files.
- Docs: project documentation.
